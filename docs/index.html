快速开始

本地运行仅需 3 个步骤

# 1. 安装依赖 pip install -e .[dev] # 2. 配置 API 密钥（可选，无密钥也能用模板模式） cp config/settings.example.yaml config/settings.yaml # 编辑 config/settings.yaml 填入 Coze/Dify/n8n 密钥 # 3. 启动 Web UI streamlit run src/testcode/web_ui.py

部署方式

选择适合你的部署方案

☁️ Streamlit Cloud 免费

最简单的方式，连接 GitHub 仓库自动部署。

  1. Fork 本仓库到你的 GitHub
  2. 访问 share.streamlit.io
  3. 点击 "New app" → 选择仓库 → 部署
  4. 在 Settings → Secrets 中配置 API 密钥

🐳 Docker 部署

适合自有服务器或云主机。

  1. 复制 .env.example 为 .env 并填入密钥
  2. 运行 docker compose up --build
  3. 访问 http://localhost:8501

💻 本地运行

开发调试，需要 Python 3.11+。

  1. pip install -e .[dev]
  2. streamlit run src/testcode/web_ui.py
  3. 浏览器自动打开 http://localhost:8501
